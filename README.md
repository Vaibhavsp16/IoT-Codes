# IoT-Codes

This repository contains Arduino, Raspberry Pi, and C practice programs.

To view any file directly from GitHub in the terminal, run:

```powershell
curl.exe --ssl-no-revoke -L "https://raw.githubusercontent.com/Vaibhavsp16/IoT-Codes/main/[file_name]"
```

Replace `[file_name]` with the exact file name from the list below.
File names are case-sensitive.
